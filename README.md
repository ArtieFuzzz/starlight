# Starlight

## Requirements

- R2 / S3 Bucket
- Docker *(Optional)*
- Java 16+ *(I use *20*)*
- Kotlin *(Latest)*

## Setup

Firstly you'll need to generate a key @ [mkjwk.org](https://mkjwk.org/)
- EC
- P-384
- Signature
- ES384:ECD
- Key ID can be any
